# Touche Medical GitHub Pages Website

Static website for the Touche Medical GitHub Pages profile.

## Purpose

This site presents Touche Medical as a medical device electronics and embedded
software development group focused on low-power devices, sensitive biosignal
inputs, analog front ends, BLE, and STM/TI/ARM battery-powered systems.

## Publish To GitHub Pages

Create a GitHub repository named:

```text
touche-Medical.github.io
```

Then publish this folder:

```powershell
cd "C:\Users\Admin\Desktop\project\touche-Medical.github.io"
git init
git add .
git commit -m "Create Touche Medical GitHub Pages site"
git branch -M main
git remote add origin https://github.com/touche-Medical/touche-Medical.github.io.git
git push -u origin main
```

After pushing, the site should publish at:

```text
https://touche-Medical.github.io
```
